Pengaturan Gemini API

Secara default, chatbot menggunakan Local NLP Engine (tanpa kunci API). Anda dapat memasukkan kunci API Gemini untuk beralih ke AI Agent yang cerdas dan mampu bernalar layaknya asisten sesungguhnya.

Kunci API disimpan secara lokal di browser Anda dan langsung dikirim ke Google API.
